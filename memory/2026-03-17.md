## 數據源調查結果

### ✅ Westmetall.com — 主力數據源
- URL 格式：`https://www.westmetall.com/en/markdaten.php?action=table&field=LME_XX_stock`
- 字段：`LME_Cu_stock`, `LME_Zn_stock`, `LME_Ni_stock`
- 每個頁面提供：日期 | LME 庫存（噸）| Cash-Settlement（USD/t）| 3-month（USD/t）
- HTML 結構：`<tbody><tr><td>日期</td><td>庫存</td><td>現貨價</td><td>3月期</td></tr>...`
- **Cobalt 不在 Westmetall 追踪列表中**（只有 Cu/Sn/Pb/Zn/Al/Ni）
- 概覽頁同時顯示所有 LME 現貨價 + 庫存 + 每日漲跌

### ✅ Stooq.com — 備用（精度較低）
- `ZN.F` = 111.73 cents/lb（COMEX 鋅期貨，非 LME）→ ≈2,462 USD/t
- `NI.F` = 661.8 cents/lb（NYMEX 鎳期貨，非 LME）→ ≈14,586 USD/t
- 單位是美分/磅，需換算；且是 COMEX/NYMEX 非 LME → 不選用

### ❌ LME 官方 — Cloudflare 全面封鎖（HTTP 403）

### ❌ Co USD — 無免費源
- Westmetall 無 Co 數據
- LME Cobalt 是非標準小合約，Stooq/Yahoo 均無
- 建議：保持 null，等待付費 API 或用戶提供

## 代碼修改記錄

**文件**: `D:\Projects\metal-price\scripts\fetch-all-data.mjs`

### 新增函數
1. `parseWestmetallDate(str)` — "16. March 2026" → "2026-03-16"
2. `fetchWestmetallMetal(fieldName)` — 請求單個品種的 stock 表格頁，解析第一行數據

### 修改
- `fetchLmeInventory()` 方案A：原 LME 官方 API → 改為 Westmetall
  - 在 result 中附加 `_wmPrices` 供 main() 提取 USD 現貨價
- `main()` Promise.all：移除多餘的 `fetchWestmetallAll()` 調用
  - 改為 `const westmetall = inventory?._wmPrices ?? ...`

- `prices.zinc.usd` → `westmetall?.zinc?.cashUsd ?? null`
- `prices.nickel.usd` → `westmetall?.nickel?.cashUsd ?? null`
